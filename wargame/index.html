/wargame

The wargame is an ongoing competition run by the Cyber Security Club at Virginia Tech. It currently has two components: "pwn", where you compromise machines or applications on the network to retrieve a flag to earn points, and "crack", where you crack passwords from a list of hashed passwords to score points.

Originally intended for members of the Cyber Security Club, the wargame is now open to any interested party, regardless of affiliation. To prevent use of machines as attack vectors, access to the wargame network is available only through an isolated VPN. To connect, you will need a Linux machine with OpenVPN installed (BackTrack comes with this by default) and this file:

  • wargame.tar.gz

You can download it and place it wherever you'd like, then extract it with tar -zxvf wargame.tar.gz. It will create a wargame directory. To connect to the wargame, simply run cd wargame && ./connect_wargame.sh and follow the prompts. When you see "Initialization Sequence Completed", you are connected to the network.

Targets on the network will be located between 10.13.37.10 and 10.13.37.50, and passwords to crack are available on 10.13.37.10. Note that since the network is isolated, so you will not be able to access the Internet from compromised machines.

If you run into any problems or need help, feel free to contact one of our officers, either in IRC, on our listserv, or by email. Have fun!

Scores

Submit a Flag

If you would like to submit a flag to be scored, please use ctfengine to do so. ctfengine is our new experimental automatic wargame scoring engine and is still fairly rough around the edges, so please contact one of our officers if you have any problems with it.
